Radix Style is a baseline CSS framework providing a style normalisation, a grid and flair for key elements.

Our Grid System

Similar to ones you've seen with 960, Bootstrap or Foundation. 12 Colums or a %column basis.

Use the g class to create a grid defining container. A div with a class like c# will define the columns, at o# defines it's offset (if any is needed)
